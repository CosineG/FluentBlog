Fluent Blog
===========

- 学习ASP.NET Core的同时练习编写的渣作
- 采用ASP.NET Core 5.0开发的博客，基于MVC模式
- 界面使用了bootstrap + bootstrap-fluent-design，是极简、带有fluent design风格的简易博客

目前完成的功能：

- 文章（markdown支持）
- 评论（valine）
- 动态
- 归档
- 友情链接
- 简易的后台管理

接下来可能的目标：

- 改为前后端分离，使用Blazor框架
